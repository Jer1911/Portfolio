Instructions for running the program on Linux:

From the terminal, enter:

	"python team5_project1.py -i input_file_name -o output_file_name"

without the quotes, where input_file_name is replaced by the name of the file
you wish to test, and output_file_name is your preferred output file name.

The program will append '_dis.txt' to your output file's name.

For exampe, for our test we entered the following command:

	"python team5_project1.py -i test1_bin.txt -o team5_out"

without quotes, and a file 'team5_out_dis.txt' was created.
